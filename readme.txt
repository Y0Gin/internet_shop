Цель: разработать решение, которое позволит персонализировать предложения постоянным клиентам Интернет-магазина «В один клик», чтобы увеличить их покупательскую активность.

Задачи:

Разработайте модель, которая предскажет вероятность снижения покупательской активности
Выделить сегмент покупателей, проанализировать его и предложить варианты увеличения его покупательской активности.

Используемые модели:
Разработан папйплайн, осуществляющий перебор четырех видов моделей (деревья решений, метод KNN, логистическая регрессия и метод опорных векторов), их гиперпараметров, а также способов кодирования и масштабирования признаков.

Результат применения:
В результате выполнения проекта решены две поставленные задачи:
- разработана модель, которая предсказывает вероятность снижения покупательской активности на сонове Метода опорных векторов.
- выделен сегмент покупателей с высокой вероятностью снижения покупательской активности и наиболее высокой прибыльностью. Сегмент проанализирован, предложены варианты увеличения его покупательской активности.
